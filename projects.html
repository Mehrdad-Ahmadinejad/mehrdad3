Academic Projects

  • Numerical investigation of flow over a cylinder using ψ-ω formulation: Developed a finite difference code in Fortran and Matlab to simulate flow around a cylinder using a mapping method for mesh generation.
  • Investigation of laminar channel flow using the projection method: Implemented a finite difference code in Fortran and Matlab to accurately study pressure fields in laminar channel flow.
  • Heat Transfer Analysis in Various Coordinates: Developed a general boundary condition code for 1D, 2D, and 3D heat conduction in Cartesian, Cylindrical, and Spherical coordinates using Matlab and Fortran.
  • Numerical investigation of Blasius boundary layer: Solved the third-order non-linear boundary value problem using the Runge-Kutta method combined with a shooting technique.
  • Root Finding Algorithms: Developed a general bisection method code in Fortran and Matlab to compute multiple roots of arbitrary functions.
